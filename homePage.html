Physics Breaker

Physics Breaker is an app for an iOS (iPhones) that solves physics problems which the user draws out. The user is provided with a number of physics objects such as masses, inclined planes, springs, force vectors, etc. and uses these to draw out a diagram of the physics problem they want solved. It then takes in two variables (such as velocity and time) and solves for the former when the latter reaches an input value (e.g. if you wanted to know the time at which an object hit 20m/s in freefall you would enter in time for the first variable and v=20 for the second). This project is still in development.

Ultuzzlement

Ultuzzlement is the aptly named Ultimate Puzzle. While this project is only in the earliest stages of development now, these early stages will take a long, long time. The purpose of this app (which will operate on iOS) is to be the ultimate puzzle. In other words, I want it to be something that can't really be figured out. With every type of puzzle to date, there is a patter, an algorithm, a set of rules that can be followed to solve it every time. I know that this is unavoidable in a puzzle, but I want to create something that at least mimics randomosity by coming very close to it. What you'll see, should you go to this page, is a bunch of very strange ideas. Prepare to be amazed.
